Hỏi: Là một web developer ( nhà phát triển web), bạn thấy mình ở đâu trong 5-10 năm sắp tới?

Đáp: Ashish Kedia:

____

Nếu tôi là cậu, tôi sẽ ghét việc nghĩ tới bản thân vẫn là một web developer trong 5-10 năm sắp tới. Phát triển websites (những trang web), đặc biệt những trang nội dung tĩnh hoặc quản lý nội dung đơn thuần, rồi sẽ không còn là một kỹ năng xác đáng trong vài năm tới nữa.

Nó rồi sẽ giống như một kỹ năng cơ bản mà mỗi developer đều phải có. Thử nghĩ về một người ghi "thông thạo MS Word và Excel" vô hồ sơ xin việc của họ trong thời đại này xem. Điều đó có giá trị gì không? Cách đây 12-15 năm về trước thì nó từng có giá trị. Giờ thì hết rồi. Những kỹ năng đó giờ quá thông dụng và người ta trông đợi cậu phải hiển nhiên thông thạo chúng để có thể làm bất kỳ công việc văn phòng nào.

Phát triển website rồi sẽ chỉ còn đơn giản là lựa chọn bảng màu, quyết định nội dung, xong là chỉ cần để cho một công cụ tạo website làm phần còn lại. Hầu hết những công cụ này đều đã được tự động hóa. Ngay cả những thứ như triển khai các dịch vụ web có khả năng mở rộng và API (Application Programming Interface aka Giao diện lập trình ứng dụng) đã trở nên dễ dàng hơn nhờ các nền tảng điện toán đám mây được cung cấp bởi Amazon, Google, Microsoft và các dịch vụ khác.

Lời dịch giả: Số là mình đang mộng làm web developer nên cũng ráng hóng chuyện về những chủ đề liên quan. Đọc đc bài này thấy hơi hoang mang vì
1. Mình thấy trên thị trường hiện giờ đang bắt đầu xuất hiện những dịch vụ thiết kế trang web trên mạng với những dòng tít bắt mắt rằng là bạn có thể tự thiết kế riêng một trang web cho mình thông qua những bước rất đơn giản mà bất cứ ai cũng làm đc. Mình nghĩ chuyện thiết kế web menu cho nhà hàng hay những doanh nghiệp nhỏ là hoàn toàn có khả năng bị tự động hóa.
2. Không quá khó để trở thành một web developer trong thời điểm hiện tại. Không biết có phải do thị trường khác nhau không, nhưng theo thống kê của chính phủ Mỹ (https://www.bls.gov/ooh/computer-and-information-technology/mobile/web-developers.htm), phần lớn web developer chỉ cần bằng cao đẳng là đã có thể làm được công việc này. Đa phần kiến thức để trở thành 1 web developer đều có thể học được từ trên mạng internet.
3. Mình đọc thêm những câu hỏi liên quan cũng thấy có nhiều người đồng quan điểm với OP.
