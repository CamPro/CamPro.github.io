Hỏi: Bạn có thể nói về một thông điệp ẩn trong code mà chỉ lập trình viên mới hiểu được không?
Trả lời: Soroush Arghavan, Kỹ sư phần mềm (2015-nay)
.
Bạn cùng phòng của tôi có lần đưa cho tôi đoạn ghi chú này:

boolean done = false;

while (!done) {
if (Calendar.getInstance().get(Calendar.HOUR_OF_DAY) == 21) {
System.gc();
done = true;
}
}

Anh ta muốn tôi dọn rác vào lúc 9 giờ tối. 😂😂😂
