**Hỏi: Ngôn ngữ lập trình nào tốt nhất, C, C++, Python hay là Java?**

**Đáp: [Andrea Ferro](https://www.quora.com/profile/Andrea-Ferro), biết vài ngôn ngữ lập trình, đang học thêm vài ngôn ngữ nữa.**

Nếu bạn muốn viết cả một hệ điều hành, tôi khuyên bạn nên dùng C.

Nếu bạn muốn viết một ứng dụng phức tạp kinh khủng mà yêu cầu trọng yếu nhất là tốc độ xử lý, tôi khuyên bạn nên dùng C++.

Nếu thời gian ra mắt sản phẩm là chìa khóa, tốc độ xử lý không quan trọng, tôi khuyên bạn nên dùng Python.

Nếu sếp bạn bảo là: “code Java không tao đuổi việc chết bà”, tôi khuyên bạn hãy xài Java rồi nhảy việc đê.
